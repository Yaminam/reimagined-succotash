# Shreyash - brand logos (CORRECTED)

Three subfolders:
- **logos/** - original source (reference).
- **logos-corrected/** - transparent PNGs resized to website spec (800px long edge). Already LIVE on the site. *Do not upload these to ChatGPT* - the mark is a dark navy colour on a transparent background, so ChatGPT sees only a dark rectangle.
- **logos-visible/** - the same logos placed on a WHITE background so the artwork is clearly visible. **Upload these to ChatGPT.**

### Workflow
1. Open ChatGPT (image mode) and **upload the file from logos-visible/** (the 'Upload to ChatGPT' column).
2. Paste the **master prompt** + the row's short prompt (brand + target size).
3. Save ChatGPT's transparent result into logos-corrected/ (the 'Save result to' path), same filename. That folder is what the site uses.

## Master prompt

```
Attached is an official brand logo on a white background. Recreate it pixel-accurately on a FULLY TRANSPARENT background - match the exact typography, letterforms, symbol and spacing; do not redesign, simplify, recolour, or add anything. Keep the original aspect ratio: no crop, no stretch, no square padding. Export as a transparent PNG at 800px on the long edge.
```
